🎯 Max-Finder

The Second-Max Paradox

The Objective

Hidden behind the grid is an array of unique random numbers. Your goal is to find the index of the Absolute Maximum value using the fewest queries possible.

The Catch

When you query a range, the system never tells you where the Max is. It only reveals the index of the Second-Largest number in that range.

Example Query: Range [3, 7]
Hidden Values: [..., 12, 99, 45, 87, 21, ...]
Result: "Second Max is at Index 6" (Value 87)

How to Play

  • Select a Range: Click two different cells to define a search window $[L, R]$.
  • Query: Use the "Query" button. This costs 1 point. Use the revealed "Second-Max" to narrow down where the true Max is hiding.
  • The Logic: If you query a range and it returns index 'X', you know the true Max of that range is either to the left or right of 'X', or is 'X' itself if your range changes!
  • Guess: Think you found it? Select a single index and hit "Guess". Be careful—a wrong guess resets your progress!
